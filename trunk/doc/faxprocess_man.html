NAME

faxprocess.sh – sends input files as faxes

SYNOPSIS

faxprocess.sh [PATTERN] ... DIRECTORY

DESCRIPTION

Scans DIRECTORY for files matching PATTERN and sends them as faxes.

The fax number is matched by a customer ID found in the input file.

Successfuly processed files are stored in a directory named processed within

the search directory. Failed files are stored in a directory named bad within

the processed directory.


PATTERN – a matching pattern understood by the standard UNIX ls command.

If no pattern is supplied, the pattern *.spl is used.
